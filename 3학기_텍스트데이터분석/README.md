# 보험사 VOC 데이터 가명처리 기술 연구

* 산출물 : 보험사VOC데이터_가명처리기술연구.pdf

## 1. 프로젝트 개요

### 1.1 목적
* 국민대학교 대학원 텍스트데이터분석 과목 과제 수행
* 보험사 VOC(Voice of Customer) 데이터 분석을 위한 가명처리 기술 연구
* 개인정보 보호를 위한 텍스트 데이터 가명화 방법론 개발

### 1.2 연구 배경
* 고객 의견 데이터 분석의 중요성 증가
* 개인정보보호법 준수를 위한 가명처리 필요성
* 텍스트 데이터에서의 개인정보 식별 및 처리 기술 필요
* 비식별화된 데이터를 활용한 의미 있는 분석 가능성

## 2. 프로젝트 내용

### 2.1 연구 주제
보험사 VOC 데이터 가명처리 기술 연구

### 2.2 주요 연구 내용
* VOC 데이터의 개인정보 식별 및 분류
* 텍스트 데이터 가명처리 기법 연구
* 가명처리 후 데이터 품질 유지 방안
* 가명처리된 데이터의 분석 가능성 평가
* 개인정보 보호 수준과 데이터 활용성 균형

### 2.3 기술 스택
* 자연어 처리: NLTK, spaCy, KoNLPy
* 텍스트 전처리: 정규표현식, 형태소 분석
* 개인정보 식별: NER(Named Entity Recognition)
* 가명처리 알고리즘: 마스킹, 토큰화, 대체
* 데이터 분석: 텍스트 마이닝, 감성 분석, 토픽 모델링
