## 1. 프로젝트 개요

### 1.1 목적
* 국민대학교 대학원 딥러닝 과목 조별 과제 수행
* 딥러닝 기반 피부질환 분류 모델 개발
* 모바일 환경에서 대상포진(Herpes Zoster) 진단을 위한 AI 모델 구축

### 1.2 연구 배경
* 피부질환의 조기 진단의 중요성
* 모바일 헬스케어의 확산과 접근성 향상 필요
* 딥러닝 기반 이미지 분류 기술의 의료 분야 적용

## 2. 프로젝트 내용

### 2.1 연구 주제
대상포진 모바일진단을 위한 딥러닝 피부질환 분류 기법

### 2.2 주요 연구 내용
* 피부질환 이미지 데이터셋 구축 및 전처리
* 딥러닝 모델 설계 및 학습
* 모바일 환경 최적화
* 대상포진 분류 성능 평가

### 2.3 기술 스택
* 딥러닝 프레임워크: TensorFlow / PyTorch
* 이미지 처리: OpenCV, PIL
* 모델 아키텍처: CNN 기반 분류 모델
* 모바일 배포: TensorFlow Lite / ONNX
