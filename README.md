# kmu_aibd_project

국민대학교 경영대학원 AI 빅데이터학과 교과과정 중 진행한 프로젝트 모음입니다.

## 📚 프로젝트 개요

본 저장소는 국민대학교 경영대학원 AI 빅데이터학과의 학기별 교과과정에서 수행한 다양한 데이터 분석 및 머신러닝 프로젝트를 포함합니다. 각 프로젝트는 실무 중심의 데이터 사이언스 역량을 함양하기 위한 목적으로 진행되었습니다.

- **기술 스택**: Python, pandas, Streamlit
- **데이터**: 교육부 학생건강검사 데이터 (2015-2022)

## 📊 프로젝트별 주요 성과

1. **공공데이터 분석**: 교육부 학생건강검사 데이터를 활용한 종합적인 EDA 및 대시보드 구축
2. **AutoML 활용**: H2O AutoML을 통한 최적 모델 자동 선정 및 성능 평가
3. **A/B 테스트**: 통계적 검정을 활용한 웹 UI 개선 효과 검증
4. **딥러닝 모델**: 피부질환 분류를 위한 CNN 기반 모델 제안
5. **텍스트 분석**: 개인정보 보호를 고려한 가명처리 기술 연구
6. **추천시스템**: 그래프 신경망 및 협업 필터링 기반 추천 모델 구현
7. **클라우드 ML**: AWS SageMaker를 활용한 ML 파이프라인 구축
